# Rucio Containers

This repository provides the official container images for [Rucio](https://rucio.cern.ch/). It contains Dockerfiles and supporting build assets for the full Rucio stack — including the server, daemons, clients, init jobs, web UI, and several test containers. Released images are published to the [rucio organization on Docker Hub](https://hub.docker.com/u/rucio) and tagged in lockstep with upstream Rucio releases. Contributions and issues are welcome; the project is licensed under Apache-2.0.
